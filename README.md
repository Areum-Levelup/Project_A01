# Project_A01
레벨업 동아리 첫 번째 프로젝트

## Description
아름고등학교 레벨업 동아리에서 개발한 Unity 프로젝트입니다. 이 프로젝트는 "아포칼립스 능력자 배틀" 장르에 속합니다. 아직 개발에 사용할 Unity 버전은 정해지지 않았으며, 개발팀은 총 4명으로 구성되어 있습니다.

## Features (예정)
- 매력적인 게임 플레이 메커니즘 (예정)
- 다양하고 독특한 전투 능력 (예정)
- 아름다운 시각적 효과와 애니메이션 (예정)
